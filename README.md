## Quick Start

```bash
git clone https://github.com/zektrace/portfolio.git
cd portfolio
pnpm install
pnpm dev
```

---

## Commands

| Command | Description | Port |
|---------|-------------|------|
| `pnpm dev` | Start development server | 9999 |
| `pnpm dev:host` | Start dev server (network accessible) | 9999 |
| `pnpm build` | Build for production |  |
| `pnpm start` | Preview production build | 4173 |
| `pnpm start:host` | Preview production (network accessible) | 4173 |
| `pnpm clean` | Remove build artifacts | |
| `pnpm rebuild` | Clean + build |  |
